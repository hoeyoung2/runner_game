#  runner_game
pygame을 통해 게임만들기
## 파이썬응용 팀 프로젝트

 맵 안에서 플레이어가 달리면서 item (코인, 보물상자 등) 을 찾아 획득하고 enemy들을 피해서 도착 지점까지 가는 게임

	

  > 팀원 
   
   정나영, 이예진, 이회영, 임유정

    
 > 실행환경
 
  Visual Stdio Code,  Python 3.10
    
    
  > 프로젝트 특징
    
  수업에서 학습한 pygame에서 나아가 타일을 통해 게임을 디자인할 수 있도록 pyTMX모듈을 이용하였다.
